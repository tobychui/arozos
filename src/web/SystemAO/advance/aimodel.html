Connect ArozOS AGI scripts to any OpenAI-compatible chat completion endpoint, define per-model pricing and track token usage & cost.

Connection

OpenAI-compatible base URL. Examples: https://api.openai.com/v1, https://openrouter.ai/api/v1, http://localhost:11434/v1 (Ollama). The path /chat/completions is appended automatically.
Sent as Authorization: Bearer <key>. Leave blank to keep the saved key.

Using it from an AGI script

requirelib("aimodel");

// Plain text prompt
var answer = aimodel.chat("Summarize the theory of relativity in one sentence.");

// With options (override model / add a system prompt)
var reply = aimodel.chat("Translate to French: good morning", {
    model: "gpt-4o-mini",
    system: "You are a professional translator.",
    temperature: 0.2
});

// File based (image for vision models, or a text document)
var caption = aimodel.chatWithFile("Describe this image", "user:/Photo/cat.png");
var review  = aimodel.chatWithFile("Review this code", ["user:/Desktop/main.go"]);

// Low-level call returning the full response (incl. usage)
var res = aimodel.request([{role:"user", content:"Hello"}], {model:"gpt-4o-mini"});
console.log(res.usage.total_tokens);

// Current aggregated usage / configured models
var usage  = aimodel.usage();
var models = aimodel.models();
