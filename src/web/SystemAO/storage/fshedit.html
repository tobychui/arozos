S3 / S3-Compatible Storage

Path format:

  • s3.amazonaws.com/my-bucket — AWS S3
  • s3.us-west-2.amazonaws.com/my-bucket/prefix — AWS S3 with region & prefix
  • http://192.168.1.100:9000/my-bucket — MinIO (local, no SSL)
  • s3.wasabisys.com/my-bucket — Wasabi
  • <account>.r2.cloudflarestorage.com/my-bucket — Cloudflare R2

Prepend http:// to disable SSL (e.g. local MinIO). HTTPS is used by default.

Leave Username / Password field empty for using the old config
