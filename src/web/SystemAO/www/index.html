Web Root Document

This is the root document of the home page service for this ArozOS Host.
To browse a user's homepage, enter the username as subpath for the URL (i.e. www/username)

To shutdown the homepage feature, set -enable_homepage flag to false from your startup script and restart.
All users that are not logged in will be redirected to login page instead.

To change this file, edit web/SystemAO/www/index.html
