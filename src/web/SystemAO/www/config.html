Personal Home Page

Host a static website on your ArozOS

About Personal Home Page
  • You can host a static website directly from your ArozOS user storage.
  • All website files are accessible through your File Manager inside your user file system.
  • Your home page is publicly accessible at http:///www/{username}/
Settings
Enable Personal Home Page
Make your web root publicly accessible via the /www/ path.
Web Root Folder
The folder whose contents will be served as your home page.
All files inside the web root are publicly accessible to everyone on the internet.

* Changes are saved automatically.
