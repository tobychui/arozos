How It Works

When enabled, the scheduler calls cron.agi every minute on your behalf. The script records a timestamp and run count using appdata storage, which you can see above after each tick.

// cron.agi — runs every minute requirelib("appdata"); appdata.set("CronDemo", "lastRun", new Date().toISOString()); var n = parseInt(appdata.get("CronDemo","runCount")||"0",10); appdata.set("CronDemo", "runCount", String(n + 1)); sendOK();
