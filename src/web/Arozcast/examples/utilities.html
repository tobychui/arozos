notifyTakeover()

Broadcasts an arozcast.takeover BroadcastChannel message. Other open sender tabs (Musicify, Movie, Photo, or other instances of this SDK) will receive the takeover event and yield.

destroy()

Releases all SDK resources: removes event listeners, closes the BroadcastChannel, closes the WebSocket silently (no media.stop). Called automatically on page unload. Use it to tear down the instance while the page is still open.
