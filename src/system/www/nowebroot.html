You are seeing this page because you have enabled home page on your account.

The ArozOS Home Page is a simple way to host your files using a web page so you can access it anywhere without login.

However, it seems you haven't configure your web root location and hence, this error occured.

How can I set it up correctly?

Login to your Web Desktop System and open System Setting → Network & Connection → Personal Page and configure the web root accordingly.
