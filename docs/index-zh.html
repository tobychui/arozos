為何選用 ArozOS?

雲端桌面

簡單又優美的網頁桌面界面，讓你可以輕鬆完成日常生活上的簡單工作

為「樹莓派」而設計

系統適用於 樹莓派 等資源有限的單片電腦，也能在一般電腦和伺服器上執行

開源

ArozOS 是一個開源項目，並開放免費作個人或教學使用

ArozOS主機
網頁桌面系統

快速讓任何安裝了瀏覽器和連接互聯網的電腦變成屬於你的個人裝置，隨時隨地存取你在雲端上的檔案。

  • 無需要特別的使用者權限
  • 支援 Chrome， Firefox 及 Safari 瀏覽器
  • 可透過網頁桌面、FTP 及 WebDAV 存取檔案
  • 可能是世界上最好的網頁檔案管理員
  • 基於 HTML5 的現代化 WebApps


ArozOS手機
網頁桌面系統

使用 PWA 技術，讓你的桌面也能在手機上開啟，讓裝置之間傳送檔案更加方便快捷。

  • 無需安裝任何 App
  • 支援 Android 及 iOS 裝置
  • 支援 Chrome 及 Safari
  • 類視窗型作業模式
  • 可在多台裝置之間進行即時同步

豐富的內置功能

應有盡有的各種 WebApp 功能，讓您在網頁上擁有桌面般的使用者體驗

簡單而潔淨

仿玻璃和圓角風格的視窗，簡單直接的 WebApp 命名方式

支援多媒體

內置音樂、圖片及音樂播放器，適合用作多媒體串流之用

檔案管理員

可能是你能在網頁桌面環境裡找到的，最好的一個檔案管理員

簡單易用

支援在筆記本電腦、手機等上的拖拉操作

Git 安裝

開源的 WebApp 模組可以透過內置的 模組安裝器，使用 Git Clone 方法下載

快速連接

支援 Windows 、Mac 跟 Linux 透過網頁界面、FTP 或 WebDAV 連接到 ArozOS 主機

開源

有不合用的地方？自己隨便改寫吧！

使用 Go 語言開發

比起其他語言如 PHP，Go 更能榨乾樹莓派的處理能力，使系統運行效率更佳
