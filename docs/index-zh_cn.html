为何选用 ArozOS?

云端桌面

简单又优美的网页桌面界面，让你可以轻松完成日常生活上的简单工作

为「树莓派」而设计

系统适用于 树莓派 等资源有限的单片电脑，也能在一般电脑和伺服器上执行

开源

ArozOS 的部分组件是开源项目，并在 Github 开放免费使用

ArozOS主机
网页桌面系统

快速让任何安装了浏览器和连接互联网的电脑变成属于你的个人装置，随时随地存取你在云端上的档案。

  • 不需要特别的使用者权限
  • 支持 Chrome， Firefox 及 Safari 浏览器
  • 可透过网页桌面、FTP 及 WebDAV 存取文件
  • 可能是世界上最好的网页文件管理员
  • 基于 HTML5 的现代化 WebApps


ArozOS手机
网页桌面系统

使用 PWA 技术，让你的桌面也能在手机上开启，让装置之间传送文件更加方便快捷。

  • 无需安装任何 App
  • 支持 Android 及 iOS 装置
  • 支持 Chrome 及 Safari
  • 类视窗型作业模式
  • 可在多台装置之间进行即时同步

丰富的内置功能

应有尽有的各种 WebApp 功能，让您在网页上拥有桌面般的使用者体验

简单而洁净

仿玻璃和圆角风格的视窗，简单直接的 WebApp 命名方式

支持多媒体

内置音乐、图片及音乐播放器，适合用作多媒体串流之用

文件管理员

可能是你能在网页桌面环境里找到的，最好的一个文件管理员

简单易用

支援在笔记本电脑、手机等上的拖拉操作

Git 安装

开源的 WebApp 模组可以透过内置的 模组安装器，使用 Git Clone 方法下载

快速连接

支援 Windows 、Mac 跟 Linux 透过网页界面、FTP 或 WebDAV 连接到 ArozOS 主机

开源

有不合用的地方？自己随便改写吧！

使用 Go 语言开发

比起其他语言如 PHP，Go 更能榨干树莓派的处理能力，使系统运行效率更佳
