ArozOS

Web Desktop System for my DIY Network Attached Storage // 用於我家的自組 NAS 之網頁桌面作業系統


Scroll down to learn more // 向下滑以了解更多




Full Mode // 完整版 Web Desktop // 網頁桌面

Turn any PC with browser into your Web Desktop client. Access all your files anytime, anywhere. // 把任何有瀏覽器之電腦變成您的網頁桌面客戶端，隨時隨地存取您的資料和檔案

No Setup and special user permission // 無需設定、安裝本地應用程式或特別使用者權限
Support Chrome, Firefox and Safari // 支援 Chrome，Firefox 及 Safari
Access via Web UI, FTP, SFTP, WebDAV or SMB // 使用網頁界面、FTP、SFTP、WebDAV 及 SMB 存取檔案
One of the best Web based File Manager // 可能是其中一個最好用的網頁檔案管理員
Easily embed other WebApps via external URLs // 輕鬆使用外部 URL 嵌入其他 WebApp

Mobile // 手機版 Web Desktop // 網頁桌面

Supporting web desktop on mobile using PWA technology, allowing file access between devices much easier. // 手機版網頁桌面採用 PWA 技術，讓您可以簡單快速地在裝置之間傳送檔案

No App Installation Required // 無需安裝任何 App
Support Android and iOS // 支援 Android 及 iOS 裝置
Works on Chrome and Firefox // 支援 Chrome 及 Firefox
Layered Window Design // 多層視窗型作業設計
Synchronize between devices // 即時在多台裝置之間進行同步




Features // 特色功能

Well Designed UI // 設計感的界面
Probably the most straight forward and pretty original web-desktop UI you could find // 可能是您能找到，最簡單直接的原創網頁桌面界面
Multimedia Playback // 多媒體播放
Build in audio and video WebApp for realtime playback and transcoding. // 內置音樂及影片播放器，並可透過網頁界面進行即時編碼串流。
File Manager // 檔案管理器
Easy to use and straight forward File Manager. Support drag & drop upload. // 簡單易用的檔案管理器，可以直接把檔案拖放進去上傳
Multiple Authentication Methods // 支援多種登入方式
Support password, oAuth and LDAP login with external account managements. // 可以使用帳戶 + 密碼、oAuth 及 LDAP 登入
Support RAID via mdadm // 支援 mdadm RAID
Support creating and expanding multiple RAID type using mdadm, including RAID0, 1 and 5 // 支援使用 mdadm 建立和擴充多種 RAID 類型，包括 RAID0、1 和 5
Virtual File System & Storage Pool Architecture // 虛擬檔案層與儲存池架構
Virtual File System layer is implemented to support mounting a remote file system into ArozOS without FUSE // 虛擬檔案系統層支援在無 FUSE 的情況下掛載遠端檔案系統到網頁桌面使用
One-Click Web Share // 檔案網絡分享
Sharing a file via URL is as simple as right-click a file in File Manager and select "Share". Support multiple access permissions! // 在檔案管理器對檔案右鍵就能快速分享檔案，並支援多種權限設定。
Create Network Share // 建立網絡分享
Setup WebDAV, FTP, SFTP or SMB (using Samba) network shares with one click! // 按一下就能設定 WebDAV、FTP、SFTP 或 SMB（需安裝 Samba）網路共用！
Scriptable Plugin System // 可編程插件系統
Scriptable plugin system based on ECMA5 (JavaScript like), support both front-end and backend extensions // 可透過 ECMA5 （類似 JavaScript）對 ArozOS 前後端進行插件編程開發。


Technical Specifications // 技術規格

Category // 分類 Specifications // 功能及規格
User Interface // 使用者界面
  • Web Desktop Interface // 網頁桌面界面
  • Mobile Desktop Interface // 手機桌面界面
  • Management Gateway (Enterprise Use) // 管理入口（企業用）
  • Scriptable Interface Module (Enterprise Use) // 可編程介面模組（企業用）
Media & Files Processing // 媒體及檔案處理
  • File Manager // 檔案管理員
  • Media Playback, Open or Editing WebApps (Audio, Video, Photo, MDEditor etc) // 多媒體檔案播放、開啟或編輯應用程式 (Audio, Video, Photo, MDEditor 等)
  • File Upload (Form POST / WebSocket Chunked) // 檔案上傳（Form POST 及 WebSocket Chunked 模式）
  • File Operations Management System with Real-time Progress, Pause and Cancel Options // 具有即時進度，並可暫停及取消之檔案操作系統
  • File Versioning Tool // 檔案版本記錄工具
  • Video Transcode (360p, 720p, 1080p, original) // 影片編碼串流（360p, 720p, 1080p 或 原解像度）
  • Media File Metadata Extraction (e.g. Audio & Video thumbnail) // 媒體檔案詮釋資料（Metadata）提取（如音樂及影片之縮圖）
  • Serverless and task scheduler. Automation tasks scriptable with ECMA5 / JavaScript // Serverless 及任務調度程序模組，可使用 ECMA5 / JavaScript 編程
Networking // 網絡與連接
  • Web Server // 網頁伺服器
  • mDNS discovery and SSDP broadcast // mDNS 服務發現 及 SSDP廣播
  • UPnP Port Forwarding // UPnP 連接埠轉送
  • WiFi Management (Support wpa_supplicant or nmcli) // WiFi 管理（支援 wpa_supplicant 或 nmcli）
  • Network File Share // 網絡檔案分享
    • WebDAV
    • SFTP
    • FTP
    • SMB (Samba)
    • Directory Server (Compatibility Mode) // 資料夾列表伺服器（相容性模式）
  • Multi Permisson Web File Sharing // 多權限模式網頁檔案分享
    • Public // 公開
    • Same user group / organization // 與分享者同一權限群組或機構
    • Logged in user // 任何已登入之用戶
    • User defined users // 分享者自定義用戶
    • User defeined group / organization // 分享者自定義之權限群組或機構
Disks Management // 磁碟管理
  • Supported Mounting File Systems // 支援掛載檔案系統
    • EXT4
    • NTFS
    • FAT (exFAT, FAT32)
    • WebDAV
    • FTP
    • SFTP
    • SMB
  • Physical Disk Automount (delay start) // 物理硬碟掛載（延遲啟動）
  • ArozOS Virtual File System // ArozOS 虛擬檔案系統層
  • Auto-cleaning tmp Folder // 自動清理之 tmp 資料夾
Security // 安全性
  • oAuth
  • LDAP
  • Conncetion & Login Log // 連接及登入記錄檔
  • IP White / Blacklist // IP 黑/白名單
  • Exponential login time (when password incorrect) // 指數登入時間（密碼錯誤時）
  • Auto Login Token (Kiosk Mode) // 自動登入權杖（Kiosk 模式）
Extensibility // 擴展性
  • ECMA5 (JavaScript like) scripting interface // ECMA5（類似 JavaScript）插件編程 API
  • 3rd party Go / Python module development with sub-service reverse proxy // 使用子服務反向代理進行第 3 方 Go/Python 模組開發
Operation Requirements // 系統需求
  • 512MB system memory and 1.5GB system storage // 512MB 記憶體及 1.5GB 磁碟空間
  • Debian based Linux Distro or Windows // 基於 Debian 之 Linux 發行版或 Windows
  • Require FFmpeg, Samba (Linux only), nmcli or wpa_supplicant (Device with WiFi only) // 需裝有 FFmpeg，Samba（在 Linux 上運行時），nmcli or wpa_supplicant（具有 WiFi 功能之裝置）
  • Recommend using mobile device with Progressive WebApp (PWA) support // 建議使用支援 PWA 的手機
