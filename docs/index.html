Why ArozOS?

Cloud Desktop

Simple and elegant web desktop interface for your daily tasks

Design for Raspberry Pi

System compatible with many brands of single board computers

Open Source

ArozOS's components is open source on Github

ArozOSPC
Web Desktop

Turn any PC with browser into your Web Desktop client. Access all your files anytime, anywhere.

  • No Setup and special user permission
  • Support Chrome, Firefox and Safari
  • Access via Web UI, FTP or WebDAV
  • One of the best Web based File Manager
  • WebApps powered by HTML5


ArozOSMobile Web Desktop

Supporting web desktop on mobile using PWA technology, making file access between devices much easier and smooth.

  • No App Installation Required
  • Support Android and iOS
  • Works on Chrome and Safari
  • Window based operations
  • Synchronize between devices
