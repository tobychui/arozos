ArozOSを選ぶ理由？

クラウドデスクトップ

シンプルでエレガントなWebデスクトップインターフェースで、日常のタスクを実行できます。

クロスプラットフォーム

ほぼすべてのデバイスに展開できます。ARM SBCやAndroidの携帯電話からx86_64 PCまで対応しています。

オープンソース

ArozOSのコンポーネントはGithubでオープンソースで公開されています。

ArozOS PC
Webデスクトップ

ブラウザを備えた任意のPCを、あなたのWebデスクトップクライアントに変えることができます。いつでもどこでもすべてのファイルにアクセスできます。

  • 設定不要、特別なユーザー権限不要
  • Chrome、Firefox、Safariに対応
  • Web UI、FTP、WebDAVを介してアクセス
  • クラウド上で最高のファイルマネージャー
  • HTML5で動作するWebアプリケーション


ArozOSモバイルWebデスクトップ

PWA技術を使用してモバイルでのWebデスクトップをサポートし、デバイス間のファイルアクセスをより簡単かつスムーズにします。

  • アプリのインストールは不要です
  • AndroidとiOSに対応
  • ChromeとSafariで動作
  • ウィンドウベースの操作
  • デバイス間の同期

素晴らしい機能

多種多様なWebアプリケーションを備えた、本格的なデスクトップ体験

シンプルで美しい

グラス調で角を丸くしたウィンドウに、直感的なWebアプリ名が表示されます。

マルチメディア

マルチメディアストリーミングに適した、組み込みの音楽、写真、オーディオWebアプリケーション

ファイルマネージャ

WebデスクトップOSで見つけることのできる最高のファイルマネージャの一つ

使いやすい

タッチパッドやタッチスクリーンを使用したドラッグ&ドロップ操作のサポート

Gitインストール

ビルドインモジュールインストーラを使用してgitクローンを介してモジュールをインストールできます

簡単接続

Web、FTP、およびWebDAVをサポートし、どのオペレーティングシステムでも簡単にアクセスできます

オープンソース

変更して、変化するニーズとユースケースに合わせることができます

Goで書かれています

システムリソースを効果的に利用しながら、ポータビリティを維持できます
